# Claude / Gemini / Copilot Prompt — StockWhy: 기간 선택 수급(외국인/기관) 추세 분석 차트 설계 + API 스펙

너는 시니어 프론트엔드/풀스택 개발자이자 데이터 시각화 UX 전문가다.
목표는 "기간 선택으로 외국인/기관 수급 추세를 주가와 함께 직관적으로 보여주는 기능"을 Next.js 14(App Router) 기반으로 구현하는 것이다.
결과물은 실제 개발에 바로 들어갈 수 있도록 **UI/데이터/백엔드/API/캐싱/제약사항**까지 포함해야 한다.

---

## 1) 제품 요구 (유저가 보는 흐름)
- 홈에서 종목 검색 → `/stock/[code]` 이동
- 종목 페이지에서 기간 선택: `1W / 1M / 3M / 6M / 1Y / Custom`
- 선택 기간에 대해:
  - 상단 KPI 요약 (외국인/기관 순매수 합계, 연속 순매수 일수, 동기간 주가 수익률)
  - 차트 2패널 구성:
    1) **주가 라인 + 외국인/기관 순매수 막대 오버레이**
    2) **공매도 변화율 + 신용잔고 비율 라인** (별도 패널)
  - 뉴스/공시 타임라인 마커는 추후 확장 가능하지만, 현재는 구조만 고려

---

## 2) 구현 제약 (중요)
- KIS 기간별 시세 API는 “한 번 호출 최대 100건” 같은 제한이 있으므로,
  - 긴 기간(예: 1Y=약 250거래일)은 **분할 호출** 또는 **주/월 단위 다운샘플링** 전략이 필요
- API rate limit을 고려하여 서버에서 캐싱을 적극 적용해야 한다.
- 클라이언트는 차트에 바로 꽂을 수 있는 **통합 시계열 배열**만 받도록 설계한다.

---

## 3) 원하는 최종 산출물
아래 내용을 순서대로 작성해라.

### A. UX/UI 설계
- 기간 선택 UI (Preset + Custom)
- KPI 카드 구성 항목과 계산 정의
- 차트 구성(축/범례/토글/툴팁/스케일)
- 긴 기간일 때 차트 가독성을 위한 샘플링 전략(일/주/월)
